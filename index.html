pxr();

a Sass function to convert px to rem with direction sensitivity

PXR Documentation

convert px to rem

you can convert values from px to rem easily,just type the px value between brackets.

Sass:input

height: pxr(32);

margin: pxr(8 16 8 32);

Css:output

height: 2rem;

margin: 0.5rem 1rem 0.5rem 2rem;

PXR Directions sensitivity

pxr is sensitive to directions,set $direction variable to ltr or rtl in your sass file,and the function will match the directions in css file output.

Sass:input

                        $direction :ltr;
                        body{
                            height : pxr(32);
                            margin : pxr(8 16 8 32);
                            padding: pxr(8 16);
                        }

Css:output if $direction:ltr;

                            body {
                                height : 2rem   ;
                                margin : 0.5rem 1rem 0.5rem 2rem;
                                padding: 0.5rem 1rem  ;
                              }

Css:output if $direction:rtl;

                            body {
                                height : 2rem   ;
                                margin : 0.5rem 2rem 0.5rem 1rem;
                                padding: 0.5rem 1rem  ;
                              }

PXR with Floats

you can also set floats property with pxr.

Sass:input

float: pxr(left);

Css:output if $direction:ltr;

float: left;

Css:output if $direction:rtl;

float: right;

Sass:input

float: pxr(right);

Css:output if $direction:ltr;

float: right;

Css:output if $direction:rtl;

float: left;

CSS Direction Property

pxr works with css direction property too.

Sass:input

direction: pxr(ltr);

Css:output if $direction:ltr;

direction: ltr;

Css:output if $direction:rtl;

direction: rtl;

Sass:input

direction: pxr(rtl);

Css:output if $direction:ltr;

direction: rtl;

Css:output if $direction:rtl;

direction: ltr;

PXR Fixed Directions

to fix a css's property direction and make it insenitive to $direction variable,you can set 'fx' parameter.

Sass:input

                            $direction :rtl;
                            body{
                            
                                //with out fx parametr
                                float    :  pxr(left);
                                margin   :    pxr(8 16 8 32);
                                padding  :   pxr(8 16 32);
                                direction: pxr(ltr);
                            
                                //with fx parameter
                                float    :     pxr(left , fx );
                                margin   :    pxr(8 16 8 32 , fx );
                                padding  :   pxr(8 16 32 , fx );
                                direction: pxr(ltr , fx );
                            }

Css:output 

                            body {

                                //with out fx parametr
                                  float    : right;
                                  margin   : 0.5rem 2rem 0.5rem 1rem;
                                  padding  : 0.5rem 1rem 2rem ;
                                  direction: rtl;
                                
                                //with fx parameter
                                  float    : left;
                                  margin   : 0.5rem 1rem 0.5rem 2rem;
                                  padding  : 0.5rem 1rem 2rem ;
                                  direction: ltr;
                                }

Using Another Unites

pxr mainly converts px to rem,but if you want to use another unites without converting values,you can do that by setting the unite parameter.

Sass:input

height : pxr(20 , vw );

margin : pxr(8 16 8 32 , px );

padding: pxr(8 16 8 , vh );

Css:output if $direction:ltr;

height : 20vw;

margin : 8px 16px 8px 32px;

padding: 8vh 16vh 8vh;

Css:output if $direction:rtl;

height : 20vw;

margin : 8px 32px 8px 16px;

padding: 8vh 16vh 8vh;

FX Parameter With Unites

'fx' parameter also works with the other unites.

Sass:input

                                $direction :rtl;
                                body{
                                
                                    //with out fx parametr
                                    margin :    pxr(8 16 8 32 , px);
                                    padding:   pxr(8 16 32 , vh);
                                    height :    pxr(20 ,vw);
                                
                                    //with fx parameter
                                    margin :    pxr(8 16 8 32 , px , fx );
                                    padding:   pxr(8 16 32 , vh , fx );
                                    height :    pxr(20 ,vw , fx );
                                }

Css:output 

                                body {

                                    //with out fx parameter    
                                    margin : 8px 32px 8px 16px;
                                    padding: 8vh 16vh 32vh ;
                                    height : 20vw   ;
                                  
                                  //with fx parameter
                                    margin : 8px 16px 8px 32px;
                                    padding: 8vh 16vh 32vh ;
                                    height : 20vw   ;
                                  }
